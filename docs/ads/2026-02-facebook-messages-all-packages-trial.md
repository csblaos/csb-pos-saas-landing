# Ad Campaign: Facebook Messages - 14-Day Free Trial (All Packages)

## 1) Campaign Brief

- Campaign Name: `FB-MSG-TH-TRIAL-ALLPKG-2026-02`
- Date: 2026-02-19
- Owner: Growth Team
- Platform: Facebook / Instagram (Meta)
- Objective: Messages
- Budget per day: THB 1,500
- Flight Dates: 2026-02-20 to 2026-03-05

## 2) Audience

- Primary audience: เจ้าของร้านออนไลน์ที่ขายผ่าน Facebook/WhatsApp/TikTok
- Secondary audience: แอดมินเพจ, พนักงานแพ็กของ, เจ้าของ SME
- Locations: Thailand (Bangkok, Chiang Mai, Khon Kaen, Hat Yai)
- Age range: 22-45
- Interests / Behaviors: E-commerce, Facebook Page Admin, COD delivery, Online seller tools
- Exclusions: คนที่เป็นลูกค้าปัจจุบัน, ทีมภายใน

## 3) Offer and Hook

- Main offer: ทดลองฟรี 14 วัน ทุกแพ็กเกจ
- Proof points: รวมออเดอร์หลายช่องทางในหน้าจอเดียว, เช็กสต็อกเรียลไทม์, ติดตาม COD ได้
- Urgency element: รอบ onboarding ฟรีมีจำนวนจำกัดต่อสัปดาห์
- CTA: ส่งข้อความเพื่อเริ่มทดลอง

## 4) Creative Plan

- Format: 1 วิดีโอสั้น + 2 ภาพนิ่ง
- Size:
  - 1:1 (Feed)
  - 4:5 (Feed mobile)
  - 9:16 (Stories/Reels)
- Visual concept: ออเดอร์ไหลจาก Facebook -> WhatsApp -> Walk-in เข้าหน้า POS เดียว
- Thumbnail concept: ข้อความใหญ่ "ทดลองฟรี 14 วัน ทุกแพ็กเกจ"
